QuickBox(快捷方式管理器)
===================


这是一个windows平台下面的快捷启动、快捷方式管理、热键管理软件，网上有很多这样的软件（音速启动、小Q桌面等）

但是这些软件不太适合自己的操作习惯，就自己开发了一个
